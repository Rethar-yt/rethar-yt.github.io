1.Python Program

This command-line program allows users to calculate the value-added tax (VAT) or sales tax on a given amount. Users can also add or deduct the tax amount from the entered amount, providing flexibility for different tax scenarios.

Features:

  • Calculate VAT or sales tax based on a specified percentage.
  • Add the tax amount to the entered amount to get the inclusive total.
  • Deduct the tax amount from the entered amount to get the exclusive total.

Skills Demonstrated: Object-Oriented Programming (OOP), Python syntax and semantics, user input validation, basic arithmetic calculations, command-line interface, exception handling, time module usage, string manipulation and formatting, looping and conditional statements.

A graphical user interface (GUI) version of this program is also available. It provides a user-friendly interface for easy input and interaction. You can find the GUI version on my GitHub repository, along with the command-line version and other projects.

Check out the code on GitHub: Universal VAT/Sales Tax Calculator

3.Python Program

This is a command line game where the player has 30 seconds to guess as many pronouns as possible. The cards are generated through various techniques in Python namely lists, dictionaries, and opening a file with the `open` function. The program also features a timer function and a dice roll function. Follow the link for more details.

Features:

  • Dice rolling to determine player position on the board.
  • Different colored cards representing various difficulty levels: yellow (easy), blue (easy to medium), and red (medium to hard).
  • Randomly generated items based on the card's difficulty level.
  • Timer function with a countdown of 30 seconds for each card.
  • Option to quit the game at any time.

Skills Demonstrated:

Python programming language, Usage of modules: time and random, Object-oriented programming (OOP) principles, Input validation and error handling, File handling for reading card data from a text file, Looping and conditional statements, String manipulation and formatting.

Check out the code on GitHub: 30 Seconds Cmd Game
